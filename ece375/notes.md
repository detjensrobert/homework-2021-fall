# ECE 375 Comp Arch & Assembly

## week 1

### ISA:

4 dims of isa:
  - operations
  - operands
  - operand locations
  - operand type / size

creation criteria:
  - functionally complete
  - power efficiency
  - simplicity

instruction categories:
  - data transfer
  - arithmetic
  - logic ops
  - control transfer
  - i/o
  - optional ones:
    - MMU / system calls
    - FPU
    - decimal math
    - string ops
